# Alert

How do you show `"hello"` to the user without asking for any input?

---

<details>
<summary><strong>A: </strong> <code>alert'hello'</code></summary>
<br>

✖ Nope.

In JavaScript, the arguments to a function must be surrounded by parenthesis
`( )`

</details>

</details>

code.

</details>
